## Sat-a-llite project
A simple website that helps you learn what satellite data analysis is and why it's important in our daily lives.

You can also try creating a collage from several satellite images of a selected area to test your hand at
analyzing your immediate surroundings and the changes occurring within them, captured by these space-based devices.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

Made by satellite and space technologies passionate
